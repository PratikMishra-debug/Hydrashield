# 📖 Overview

HydraShield is a modern disaster management platform designed to help citizens and authorities respond quickly during flood emergencies.

The project combines weather information, flood risk analysis, location-based mapping, and emergency services into a single easy-to-use web application.

This project is being developed as a portfolio-grade software engineering project with future AI and cloud integration.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/PratikMishra-debug/Hydrashield.git
```

Go into the project

```bash
cd Hydrashield
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
